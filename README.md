
Cotizador de Seguros – Demo Web

Página web estática desarrollada con HTML, CSS y JavaScript.
Permite cotizar 2 tipos de seguros utilizando información proveniente de bases de datos en formato Excel.

Este demo muestra el funcionamiento de un cotizador sencillo de seguros.
La página carga datos desde una base en Excel (convertida a `.rar` debido al límite de tamaño de GitHub) y permite al usuario:

- Seleccionar un tipo de seguro
- Visualizar coberturas
- Ingresar datos para realizar una cotización
- Ver los resultados de manera dinámica en pantalla

Todo el funcionamiento ocurre del lado del cliente, es completamente estático.



Tecnologías utilizadas:

HTML5 – Estructura del sitio
CSS3 – Diseño, estilos y maquetado
JavaScript – Lógica del cotizador y manejo de datos
Excel- Base de datos original (comprimida en `.rar` por tamaño)

Archivos :

/assets          → imágenes y recursos
/CSS             → estilos de la página
/JS              → lógica del cotizador
/data            → archivos de base (Excel comprimido)
/index.html      → página principal
/cotizacion.html → sección de cotización
/contacto.html   → página de contacto
/coberturas.html → lista de coberturas

Cómo usar el proyecto:
1. Descarga el repositorio.
2. Extrae el archivo `.rar` dentro de la carpeta data/ para obtener la base de datos Excel.
3. Abre index.html en tu navegador.
4. Navega entre las secciones y prueba el cotizador.

Nota:

Este proyecto es un demo académico, no está conectado a un backend.
Los datos cargados desde Excel son únicamente para prácticas y pruebas.
